# Changelog

All notable changes to the **Tracer** brand identity and design system will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-07-18

### Added
- **Initial Branding**: Formalized the "Tracer" project identity.
- **Color Palette**: Locked primary (`#00133A`), secondary (`#2868A6`), and neutral (`#838383`) colors.
- **Design System**: Established Material 3 design tokens (radii, spacing, typography).
- **Guidelines**: Added documentation for Splash Screen and Home Screen layouts.
- **Repository Structure**: Created the `branding/` folder organization for logos, palettes, and assets.
- **Roadmap**: Defined future expansion goals including Dark Theme and Wear OS support.
